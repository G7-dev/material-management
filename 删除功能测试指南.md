# 删除功能测试指南（更新版）

## 概述

本指南用于测试优化后的删除功能，包括批量删除规格和删除整个物品。

## 测试环境

- 开发服务器已启动
- 数据库连接正常
- 至少有一个多规格物品（如白板笔）

## 测试场景

### 场景1：编辑模态框优化

**测试目标**：验证当前库存只读，阈值可调整

**测试步骤**：
1. 打开物品补货管理页面（低库存预警）
2. 点击任意物品的"编辑"按钮
3. 验证：
   - [ ] "当前库存"输入框为灰色禁用状态
   - [ ] 无法编辑当前库存数值
   - [ ] 显示提示文案："当前库存不能在此修改，请使用\"补货\"功能调整库存"
4. 修改"预警阈值"输入框的值
5. 点击"保存修改"按钮
6. 验证：
   - [ ] 提示"设置已保存"
   - [ ] 编辑模态框关闭
   - [ ] 列表数据刷新

**多规格物品测试**：
1. 找到一个多规格物品（如白板笔）
2. 点击"编辑"按钮
3. 验证每个规格都有"预警阈值"输入框
4. 修改其中一个规格的阈值
5. 点击"保存修改"
6. 验证阈值已更新

**单规格物品测试**：
1. 找到一个单规格物品
2. 点击"编辑"按钮
3. 验证显示"整体预警设置"区域
4. 修改阈值并保存
5. 验证阈值已更新

---

### 场景2：删除单个规格

**测试目标**：验证可以删除单个规格

**测试步骤**：
1. 打开物品权限管理页面
2. 找到一个多规格物品（至少2个规格）
3. 点击物品卡片上的垃圾桶图标
4. 在删除对话框中：
   - [ ] 显示所有规格列表
   - [ ] 每个规格都有复选框
   - [ ] 显示"全选"按钮
5. 选择一个规格
6. 点击"删除选中规格 (1)"按钮
7. 验证：
   - [ ] 删除按钮显示加载状态
   - [ ] 提示"1 个规格已删除"
   - [ ] 对话框关闭
   - [ ] 列表刷新，该规格已消失
   - [ ] 物品其他规格保留

---

### 场景3：批量删除规格

**测试目标**：验证可以批量删除多个规格

**测试步骤**：
1. 打开物品权限管理页面
2. 找到一个多规格物品（至少3个规格）
3. 点击垃圾桶图标
4. 在删除对话框中：
   - [ ] 点击"全选"按钮
   - [ ] 验证所有规格都被选中
   - [ ] 按钮文字变为"取消全选"
5. 取消选择其中一个规格（至少保留一个）
6. 点击"删除选中规格 (N)"按钮
7. 验证：
   - [ ] 提示"N 个规格已删除"
   - [ ] 选中的规格被删除
   - [ ] 未选中的规格保留

---

### 场景4：删除所有规格（边界测试）

**测试目标**：验证系统阻止删除所有规格

**测试步骤**：
1. 打开物品权限管理页面
2. 找到一个多规格物品
3. 点击垃圾桶图标
4. 选择所有规格（使用全选）
5. 点击"删除选中规格 (N)"按钮
6. 验证：
   - [ ] 系统阻止删除
   - [ ] 提示"至少保留一个规格，建议删除整个物品"
   - [ ] 对话框保持打开
   - [ ] 没有任何规格被删除

---

### 场景5：删除整个物品

**测试目标**：验证可以删除整个物品

**测试步骤**：
1. 打开物品权限管理页面
2. 找到任意物品（单规格或多规格）
3. 点击垃圾桶图标
4. 在删除对话框中：
   - [ ] 查看"危险操作：删除整个物品"警告区域
   - [ ] 不选择任何规格
5. 点击"删除整个物品"按钮
6. 验证：
   - [ ] 删除按钮显示加载状态
   - [ ] 提示"物资已删除"
   - [ ] 对话框关闭
   - [ ] 物品从列表中消失
7. 刷新页面
8. 验证：
   - [ ] 被删除的物品不再出现
   - [ ] 删除是永久性的（软删除后不再显示）

---

### 场景6：删除后物品不再出现（修复验证）

**测试目标**：验证删除后物品不会再次出现

**测试步骤**：
1. 打开物品权限管理页面
2. 记录当前物品数量
3. 删除一个物品（点击"删除整个物品"）
4. 等待3秒
5. 验证：
   - [ ] 物品没有重新出现
   - [ ] 物品数量减少1
6. 刷新页面
7. 验证：
   - [ ] 被删除的物品仍然不存在
   - [ ] 物品数量保持不变

---

### 场景7：多规格物品的删除选项

**测试目标**：验证多规格物品有多种删除选项

**测试步骤**：
1. 打开物品权限管理页面
2. 找到一个有3个规格的物品
3. 点击垃圾桶图标
4. 验证对话框显示：
   - [ ] 规格列表（3个规格）
   - [ ] 全选/取消全选按钮
   - [ ] "删除选中规格 (0)"按钮（初始为0）
   - [ ] "删除整个物品"按钮
5. 选择1个规格
6. 验证"删除选中规格"按钮显示"删除选中规格 (1)"
7. 点击"删除选中规格 (1)"
8. 验证只删除选中的规格
9. 再次点击该物品的垃圾桶图标
10. 点击"删除整个物品"
11. 验证整个物品被删除

---

### 场景8：错误处理和边界条件

**测试目标**：验证错误处理和边界条件

**测试步骤**：

**测试1：未选择规格时点击删除**
1. 打开删除对话框
2. 不选择任何规格
3. 点击"删除选中规格 (0)"
4. 验证：
   - [ ] 提示"请先选择要删除的规格"
   - [ ] 对话框保持打开
   - [ ] 没有任何删除操作

**测试2：快速连续点击删除按钮**
1. 打开删除对话框
2. 选择一个规格
3. 快速多次点击"删除选中规格"
4. 验证：
   - [ ] 只有第一次点击生效
   - [ ] 按钮显示加载状态
   - [ ] 没有重复删除

**测试3：网络错误处理**
1. 打开删除对话框
2. 断开网络连接
3. 点击删除按钮
4. 验证：
   - [ ] 提示错误信息
   - [ ] 对话框保持打开
   - [ ] 可以重试
